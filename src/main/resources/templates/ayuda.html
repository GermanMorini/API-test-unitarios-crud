API para realizar pruebas de CRUD

Lista de endpoints:

  • /api Muestra esta página de ayuda
  • /api/retrieve Retorna todos los registros
  • /api/retrieve/expired Retorna todos los registros vencidos
  • /api/retrieve/byName?name=NOMBRE Busca los registros que contengan 'NOMBRE' en el nombre (case-sensitive)
  • /api/retrieve/byId?id=UUID Busca el registro de id 'UUID'
  • /api/create Crea un nuevo registro, el body es el siguiente:
    
    Se puede especificar el id:
        {
            "id":"b5425ff4-df81-4c57-84ba-44144cf0c815",
            "nombre":"NUEVA ENTRADA",
            "cantidad":42069,
            "vencimiento":yyyy-mm-dd
        }

    Sino, se genera automáticamente:
        {
            "nombre":"NUEVA ENTRADA",
            "cantidad":42069
            "vencimiento":yyyy-mm-dd
        }
  • /api/update Actualiza la información de un registro. Se debe usar el primer esquema de body mostrado anteriormente, indicando el id a modificar
  • /api/delete?id=UUID Elimina un registro por su id 'UUID'
